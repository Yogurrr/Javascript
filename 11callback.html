자바스크립트 콜백 함수

개요

다른 함수의 인자로 사용되는 함수 또는 어떤 이벤트에 의해 호출되는 함수를 의미

주로, 프로그램이 실행되는 동안 지속적으로 수행해야 하는 작업이 있을 때 사용 - 비동기 프로그래밍

어떤 정보를 원하는 쪽이 정보를 제공하는 쪽에게 반복적으로 확인(요구)하는 것보다 정보를 제공하는 쪽이 정보를 원하는 쪽에게 통보하는 것이 효율적임

달걀을 삶을 때 적정 시간이 되었는지 확인해야 하거나 친구랑 놀고 있을 때 집에 가야할 시간을 확인해야 할 때 타이머/알람(콜백함수)를 이요하는 것과 동일

ES5/ES6에서 promise나 asycn/await 등이 추가되어 콜백함수의 사용은 줄이는 추세임

타이머 함수

특정 시간이 지난 후에 함수를 실행하려면 setTimer함수를 사용

특정 시간마다 함수를 실행하려면 setInterval함수를 사용
