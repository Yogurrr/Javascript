자바스크립트 조건문

개요

특정 조건에 대한 만족여부에 따라 실행할 코드를 결정하는 기능

자바스크립트는 2가지의 조건문 제공 - if, switch

코드 블록

0개 이상의 명령문을 중괄호({})로 묶은것

블록은 단독으로 사용할 수 있으나, 일반적으로 제어문이나 함수/객체 선언시 주로 사용

if문

조건을 만족할 때만 지정한 코드블록을 실행함

if (조건식) {

참일 때 실행할 명령어(들)

}

if ~ else 문

조건을 만족하면 if 아래의 코드블록을 실행하고, 만족하지 않을 경우 else 아래의 코드블록을 실행

if (조건식) {

참일 때 실행할 명령어(들)

} else {

거짓일 때 실행할 명령어(들)

}

중첩 조건문

if나 else 안에 또 다른 조건문을 포함한 조건문을 의미

단점은 코드 가독성이 떨어지므로 사용을 자제하는 것이 좋음

다중 조건문

다중 조건에 대한 만족여부에 따라 코드블록을 실행하려면 if ~ else 뒤에 if ~ else문을 더 추가함

조건식을 거짓으로 취급하는 값들

false, 0, undefined, null, NaN, 빈 문자열등은 조건식에서 참이 아닌 거짓으로 취급

switch

다중 조건 if문을 가독성 있게 작성한 조건문

switch의 표현식을 평가하여 그 값의 일치여부에 따라 case문 아래의 코드 블록을 실행하는 구조

표현식의 평가값과 일치하는 case가 없는 경우 default문의 코드블록을 실행함

단, 표현식의 평가값과 일치하는 case문이 실행되더라도 break문이 없으면, switch문이 중단되지 않고 나머지 case문을 실행하게 됨 - full through (위에서 아래로 순서대로 실행하기 때문)

switch(조건식) {

case 값1: 코드블록; break;

case 값2: 코드블록; break;

....

default: 코드블록; break;

}

javascript 한정으로 switch의 case문에 비교 연산자를 이용한 범위를 검사하는 코드를 작성할 수도 있음 (https://stackoverflow.com/questions/5619832/switch-on-ranges-of-integers-in-javascript)

if vs switch

if문의 경우 조건식 검사는 위에서 아래로 순차적으로 진행됨

운이 나빠서, 만족하는 조건식이 맨 아래에 있다면 위의 조건식 모두를 비교한 뒤에야 비로소 도달하게 됨

하지만, switch문은 평가식의 결과값을 판단하고, 해당하는 case절로 바로 찾아가기 때문에 if문보다 효율이 좋음
